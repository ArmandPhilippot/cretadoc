# Fonts

A font can be decomposed in five values:

- a family,
- a size,
- a weight,
- a letter-spacing,
- a line-height.

## Family

Cretadoc interfaces use two different font-family. The monospace version is usually used to display code samples.

<Story name="Family">
  <PreviewList isBordered wrapAfter={1}>
    <Font orientation="inline" token="font.family.regular">

    </Font>
    <Font orientation="inline" token="font.family.monospace">

    </Font>
  </PreviewList>
</Story>
