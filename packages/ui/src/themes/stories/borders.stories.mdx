# Borders

A border can be decomposed in four values:

- a color,
- a radius,
- a style,
- a width.

## Color

Color design tokens are defined separately, you can learn more on [Colors](./?path=/docs/themes-tokens-colors--borders) page.

## Radius

Cretadoc interfaces use four different shapes of borders.

## Style

Only one style of borders is used inside Cretadoc interfaces. The value of `border.style.regular` can be any CSS keyword accepted in `style` attribute.
