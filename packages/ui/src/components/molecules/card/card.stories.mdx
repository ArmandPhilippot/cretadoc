# Card

A card is commonly used to present short content about a single subject.

## Contents

A card can be composed with the following elements:

- a cover (by using an [`<Img />`](/?path=/docs/components-atoms-image--playground) component),
- a heading (by using a [`<Heading />`](/?path=/docs/components-atoms-typography-heading--playground) component),
- an excerpt,
- one or more actions (by using either a [`<Link />`](/?path=/docs/components-atoms-link--playground) component or a [`<Button />`](/?path=/docs/components-atoms-button--playground) component).
