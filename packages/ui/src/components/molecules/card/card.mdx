# Card

A card is commonly used to present short content about a single subject.

## Contents

A card can be composed with the following elements:

- a cover (by using an [`<Img />`](/docs/components-atoms-image--docs) component),
- a heading (by using a [`<Heading />`](/docs/components-atoms-typography-heading--docs) component),
- an excerpt,
- one or more actions (by using either a [`<Link />`](/docs/components-atoms-link--docs) component or a [`<Button />`](/docs/components-atoms-button--docs) component).
