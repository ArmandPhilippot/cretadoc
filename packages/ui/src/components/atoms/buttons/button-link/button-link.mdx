# Button Link

A button link is a regular link designed as a button. It is commonly used to encourage action.

<Primary />

## Kind

The button appearance can change depending on its kind.

### Primary

<Canvas of={ButtonLinkStories.Primary} />

### Secondary (default)

<Canvas of={ButtonLinkStories.Secondary} />

### Neutral

Although you can use the `neutral` kind, if you need a neutral link, you should use the [`<Link />` component](/docs/components-atoms-link--docs) instead.

<Canvas of={ButtonLinkStories.Neutral} />

## States

It can either be enabled, be disabled or indicate a loading state.

### Enabled (default)

<Canvas of={ButtonLinkStories.Enabled} />

### Disabled

<Canvas of={ButtonLinkStories.Disabled} />

### Loading

<Canvas of={ButtonLinkStories.Loading} />

## Accessibility

Sometimes, the body is not enough explicit for screen reader users. In that case, you should use the `aria-label` attribute to give a better indication about the button purpose.
