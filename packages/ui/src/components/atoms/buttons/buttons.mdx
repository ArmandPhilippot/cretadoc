# Buttons

A button is commonly used as a call to action.

## Types

Depending on the context, you can either use:

- a regular button with [`<Button />` component](/docs/components-atoms-buttons-button--docs),
- or a link style as a button with [`<ButtonLink />` component](/docs/components-atoms-buttons-buttonlink--docs).
