# List

A list can be used to display a list of data.

## Kind

Two kinds of list are possible but depending on the usage, one should be prefer over the other:

- if the data **order** in the list is **meaningless**, you should use an **unordered list**,
- if the data should be ordered, please consider using an ordered list instead.

<Canvas>
  <Story name="Unordered">
    <List>
      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
  <Story name="Ordered">
    <List isOrdered>
      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
</Canvas>

## Nested

You can use one or more lists inside a list.

<Canvas>
  <Story name="Nested">
    <List>
      <ListItem>First</ListItem>
      <ListItem>
        Second
        <List isOrdered>
          <ListItem>First nested item</ListItem>
          <ListItem>Second nested item</ListItem>
          <ListItem>Third nested item</ListItem>
        </List>
      </ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
</Canvas>

## Markers

The list items can have a marker in front of them or not:

<Canvas>
  <Story name="Marker - Visible">
    <List>
      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>

  </Story>
  <Story name="Marker - Hidden">

      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>

You can also use a custom marker either by using a [CSS keyword](https://developer.mozilla.org/en-US/docs/Web/CSS/list-style-type) or an unicode emoji for example:

<Canvas>
  <Story name="Marker - Custom: CSS Keyword">
    <List marker="circle">
      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
  <Story name="Marker - Custom: Emoji">
    <List marker={'"\\1F44D  "'}>
      <ListItem>First</ListItem>
      <ListItem>Second</ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
</Canvas>

It is also possible to mix markers by specifying a different marker on a list item:

<Canvas>
  <Story name="Marker - Custom: Mixed">

      <ListItem>First</ListItem>

## Orientation

For design purposes, a list can also be used horizontally. It can be useful to design a breadcrumbs for example.

<Canvas>
  <Story name="Orientation - Inline">
    <List hasMarker={false} isInline isOrdered spacing="xs">
      <ListItem>First</ListItem>
      <ListItem>></ListItem>
      <ListItem>Second</ListItem>
      <ListItem>></ListItem>
      <ListItem>Third</ListItem>
    </List>
  </Story>
</Canvas>

You can also use a nested list inside an inline list:

<Canvas>
  <Story name="Orientation - Inline: Nested">
