# Description List

A description list is commonly used to implement a glossary or to display metadata. It represents a list of key-value pairs.

<Primary />

## Contents

A description list is composed of `<Term />`, `<Description />` and eventually `<Group />` components.

### Single term and description

You can use a single term with a single description:

<Canvas of={DescriptionListStories.Default} />

### Multiple terms, single description

You can also use a multiple terms with a single description:

<Canvas of={DescriptionListStories.MultipleTermsSingleDescription} />

### Single term, multiple descriptions

You can also use a single term with multiple descriptions:

<Canvas of={DescriptionListStories.SingleTermMultipleDescriptions} />

### Multiple terms and descriptions

You can also combine the examples above to define multiple terms with multiple corresponding descriptions.

<Canvas of={DescriptionListStories.MultipleTermsMultipleDescriptions} />

### Group of terms and descriptions

You can also use groups to wrap each term(s)/description(s) association. They can be used for styling purposes or to improve code readability when mixing a lot of terms and descriptions.

Here an example with the above example:

<Canvas of={DescriptionListStories.GroupOfMultipleTermsMultipleDescriptions} />
