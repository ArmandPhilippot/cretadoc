# Layout

A `<Layout />` component is used to wrap the main nodes of a website. It has usually at least:

- a [`<Header />`](/docs/components-atoms-layout-header--docs),
- a [`<Main />`](/docs/components-atoms-layout-main--docs),
- and a [`<Footer />`](/docs/components-atoms-layout-footer--docs)
