An `Icon` component is commonly used for accessibility. It allows you to include some contents in your page without making them visible.

## Accessibility

The Icon component provides some attributes to adjust the accessibility. It is up to you to define if the icon should be hidden for assistive technology or if an accessible name should be given.

The accessible name can be provided with one of:

- `aria-label`,
- `aria-labelledby`,
- `title`.

You can also add a longer description with one of:

- `aria-describedby`,
- `description`.

If you provide an accessible name, you should also set the role to `img`.

If you think that the icon should be hidden then, you should use the `aria-hidden` attribute.
