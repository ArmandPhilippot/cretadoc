# Heading

A heading is used to hierarchize your contents.

## Levels

The 6 levels of heading (`h1`, `h2`, ...) are supported. Note that `1` is the hightest section level and `6` is the lower section level.

<Canvas isColumn>
  <Story
    name="Element/H1"
    args={{
      children: body,
      level: 1,
    }}
  />
  <Story
    name="Element/H2"

      children: body,
      level: 2,

## Use a fake heading

In some cases you might want to use heading styles but semantically it is not right. Then, you can use the `isFake` attribute to simulate a heading.

<Canvas isColumn>
  A h2 element:
  <Story
    name="Kind/Real"
    args={{
      children: body,
      level: 2,
    }}
  />
  A fake h2 element:
  <Story
    name="Kind/Fake"
    args={{
      children: body,
      isFake: true,
      level: 2,
    }}
  />
</Canvas>
