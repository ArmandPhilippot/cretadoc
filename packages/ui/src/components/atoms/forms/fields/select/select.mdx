# Select

A select is used for collecting user provided information from a list of options.

<Primary />

## Single choice

By default, an user can select only one option in the select menu.

<Canvas of={SelectStories.SingleChoice} />

## Multiple choices

When using `multiple`, an user can select multiple options in the select menu.

<Canvas of={SelectStories.MultipleChoices} />

You should consider using a group of [`<Checkbox />` component](/docs/components-atoms-forms-fields-checkbox--docs) or a [`<Toggle />` component](/docs/components-molecules-forms-toggle--docs) instead.
