# Label

A label represents a caption for an item in a user interface. It should be associated with an input.

<Primary />

## State

A field can be either required or not. The label reflect this state.

<Canvas of={LabelStories.IsRequired} />

You can also change the default symbol used with `isRequired`:

<Canvas of={LabelStories.CustomRequiredSymbol} />

## Color

The label can have a different color than the foreground. It could be used, for example, to indicate a valid or non-valid field.
