# Spinner

A spinner is commonly used to indicate that a content is loading.

## Accessibility

For accessibility purposes, it is preferable to set an accessible name. You can choose to place the message as a children (visually hidden or not) or to use the `aria-label` attribute for example.

## Position

When using a description as children, you can define the spinner position:

<Canvas>
  <Story name="Position/Left">
    <Spinner position="left">Loading...</Spinner>
  </Story>
  <Story name="Position/Right">
    <Spinner position="right">Loading...</Spinner>
  </Story>
  <Story name="Position/Top">
    <Spinner position="top">Loading...</Spinner>
  </Story>
  <Story name="Position/Bottom">
    <Spinner position="bottom">Loading...</Spinner>
  </Story>
</Canvas>
