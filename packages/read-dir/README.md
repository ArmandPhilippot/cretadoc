# @cretadoc/read-dir

This package provides a Cretadoc helper to walk through directories recursively.

## Install

```
npm install @cretadoc/read-dir
```

```
yarn add @cretadoc/read-dir
```

```
pnpm add @cretadoc/read-dir
```
